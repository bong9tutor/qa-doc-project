FAO_STTask_AIMoveTo
부모: FStateTreeAIActionTaskBase
역할

DisplayName
AO AI Move To
인스턴스 데이터
FAO_STTask_AIMoveTo_InstanceData
실행 위치
서버 (AIController 필요)
TargetLocation과 AcceptanceRadius를 인스턴스 데이터로 받아 AIController::MoveToLocation()을 호출합니다. EnterState에서 이동을 시작하고, Tick에서 PathFollowingComponent의 상태를 확인하여 도착 시 Succeeded를 반환합니다. ExitState에서는 이동 중이었다면 StopMovement()로 정리합니다.

AcceptanceRadiusfloatEditAnywhere도착 허용 반경 (기본값 100)
bIsMovingboolUPROPERTY현재 이동 중인지 여부. ExitState에서 정리 판단에 사용
EnterState()Override▶
흐름
AIController 유효성 확인 → TargetLocation 유효성 확인 → MoveToLocation() 호출 → AlreadyAtGoal이면 Succeeded, 성공이면 Running, 실패면 Failed
Tick()Override▶
흐름
PathFollowingComponent::GetStatus()가 Idle이면 이동 완료로 Succeeded 반환, 아니면 Running 유지
ExitState()Override▶
하는 일
bIsMoving이 true면 AIController::StopMovement()를 호출하고 플래그를 초기화합니다.
GetAIController()Helper▶
하는 일
Context Owner가 Pawn이면 GetController()로, Controller면 직접 Cast하여 AAIController*를 반환합니다.
