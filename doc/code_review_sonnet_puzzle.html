AAO_PuzzleElement
부모: AAO_WorldInteractable · IAO_Interface_PuzzleElement
역할
모든 퍼즐 요소의 추상 베이스 클래스 -- 상호작용 타입별 상태 전환과 이벤트 브로드캐스트 담당
상호작용 타입
OneTime(한 번), Toggle(토글), HoldToggle(홀딩 후 토글) 3가지 EPuzzleElementType 지원
네트워크
bIsActivated/bInteractionEnabled 복제, MulticastPlayInteractionEffect로 VFX/SFX 동기화
이벤트 전달
BroadcastPuzzleEvent()로 ActivatedEventTag/DeactivatedEventTag를 LinkedChecker에 전송
AAO_WorldInteractable을 상속하여 플레이어의 F키 상호작용을 받고, IAO_Interface_PuzzleElement을 구현하여 Checker가 초기화/활성화를 제어할 수 있습니다. OnInteractionSuccess()에서 ElementType에 따라 bIsActivated 상태를 전환하고, BroadcastPuzzleEvent()를 통해 LinkedChecker에 GameplayTag를 전달합니다. InteractionDataAsset으로 상호작용 정보를 프리셋화할 수 있고, InteractionEffectSettings로 VFX/SFX를 Delay 포함하여 재생합니다. Abstract 클래스이므로 직접 배치 불가하며, PressurePlate, RotationPuzzleElement, TorchPuzzleElement 등이 파생됩니다.

반응 모드
OneTime(한 번 변화), Toggle(태그 추가/제거 따라), HoldActive(진행도 0~1에 따라 실시간)
