추가 멤버 변수 없음 — AGameMode 기본 속성만 상속합니다.
추가 함수 없음 — AGameMode 기본 가상 함수만 상속합니다. .cpp 파일은 include 한 줄만 존재합니다.

주요 책임
SeamlessTravel 활성화, 보이스 채팅 관리, 동기화 ServerTravel, 인벤토리 보존 판정
Lobby, Stage, Rest GameMode가 공통으로 상속하는 중간 기반 클래스입니다. 생성자에서 bUseSeamlessTravel = true로 설정하여 맵 이동 시 TCP 연결이 끊기지 않도록 합니다. 보이스 채팅 시작/정지/뮤트 업데이트를 모든 클라이언트에 일괄 전파하는 함수(StopVoiceChatForAllClients, LetUpdateVoiceMemberForAllClients)를 제공합니다. ServerTravel 전 모든 플레이어의 오디오 리소스 정리를 기다리는 동기화 메커니즘(RequestSynchronizedServerTravel)을 구현하며, 이 메커니즘은 보이스 채팅 크래시 방지를 위해 도입되었습니다. 또한 맵 이동 직전 인벤토리/체력 보존 판정(HandlePlayerTravel)과 세이프존 좌표 체크(CheckPlayerInsideSafeZone) 로직을 수행합니다.
변수명타입지정자설명
CleanupCompletePlayers TSet<TObjectPtr<AAO_PlayerController>> UPROPERTY ServerTravel 동기화 시, 오디오 리소스 정리를 완료한 플레이어 컨트롤러 세트입니다. NotifyPlayerCleanupCompleteForTravel()에서 Add되며, GetNumPlayers()와 비교하여 전원 완료 시 StartServerTravel()을 호출합니다. Travel 시작/완료 시 Empty()로 초기화됩니다.
bIsTravelSyncInProgress bool (private, 기본값 false) 동기화 ServerTravel이 진행 중인지 여부 플래그입니다. RequestSynchronizedServerTravel() 진입 시 true로 설정하여 중복 요청을 차단하고, StartServerTravel() 완료 시 false로 복귀합니다.
PendingTravelURL FString (private) 동기화 대기 중인 ServerTravel 목적지 URL을 보관합니다. "?listen" 접미사가 포함된 전체 경로이며, StartServerTravel()에서 World->ServerTravel()에 전달 후 Empty()로 초기화됩니다.

로비 맵에서 플레이어 입장(PostLogin), SeamlessTravel 복귀(HandleSeamlessTravelPlayer), 퇴장(Logout)을 처리합니다. 생성자에서 PlayerControllerClass = AAO_PlayerController_Lobby::StaticClass()와 bUseSeamlessTravel = true를 설정하고, NextLobbyJoinOrder = 0으로 초기화합니다. 입장 순서(NextLobbyJoinOrder)를 관리하여 첫 번째 입장자를 호스트로 지정하고, 레디 상태(ReadyPlayers TSet)를 추적하여 호스트 외 전원 레디 시 게임 시작을 허용합니다. BeginPlay에서 GameInstance의 PassiveReset()과 GameState의 Authority_NotifyGlobalReset()을 수행합니다.
변수명타입지정자설명
ReadyPlayers TSet<TObjectPtr<AController>> UPROPERTY 레디 상태인 컨트롤러 집합입니다. SetPlayerReady()에서 Add/Remove되고, IsEveryoneReadyExceptHost()에서 호스트를 제외한 전원 레디 여부를 판정합니다. Logout() 시 퇴장 플레이어를 Remove합니다.
NextLobbyJoinOrder int32 (protected, 기본값 0) 다음에 입장하는 플레이어에게 부여할 순번입니다. 0부터 시작하며 AssignJoinOrderIfNeeded()에서 부여 후 1씩 증가합니다. UpdateNextLobbyJoinOrderFromExistingPlayers()에서 기존 플레이어의 최대 JoinOrder + 1로 재계산됩니다.

Lobby, Stage, Rest 컨트롤러가 공통으로 상속하는 중간 기반 클래스입니다. 생성자에서 CameraManagerComponent = CreateDefaultSubobject<UAO_CameraManagerComponent>("CameraManagerComponent")를 생성합니다. 보이스 채팅 시작/정지/뮤트를 Client RPC(Client_StartVoiceChat, Client_StopVoiceChat, Client_UpdateVoiceMember)로 처리합니다. 맵 이동 전 Client_PrepareForTravel RPC에서 오디오 리소스 정리(CleanupAudioResource)를 수행하고, Tick에서 IsVoiceFullyCleanedUp()을 폴링하여 완료 시 Server_NotifyReadyForTravel을 전송합니다. OnPossess와 OnRep_Pawn 모두에서 BindASCAbilityFailed()를 호출하여 서버/클라이언트 양쪽에서 AbilityFailedCallbacks를 바인딩합니다.
