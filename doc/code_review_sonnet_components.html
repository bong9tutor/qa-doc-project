UAO_PlayerCharacter_AnimInstance
부모: UAnimInstance · IAO_TraversalTransform
역할
플레이어 캐릭터의 애니메이션 상태 갱신 및 Motion Matching용 궤적(Trajectory) 생성
부모 클래스
UAnimInstance, IAO_TraversalTransform (트래버설 위치 인터페이스)
실행 위치
모든 클라이언트. NativeThreadSafeUpdateAnimation으로 워커 스레드에서 실행 가능
주요 책임
속도·가속도·상태(Gait, Stance, MovementMode) 갱신, PoseSearch 궤적 계산, 이동 분석 함수 제공
UAnimInstance를 상속하며 FAOAnimInstanceProxy를 통해 캐릭터와 MovementComponent에 대한 참조를 스레드 안전하게 보관합니다. 매 프레임 NativeThreadSafeUpdateAnimation에서 UpdateTrajectory, UpdateEssentialValues, UpdateStates를 순서대로 호출하여 속도·가속도·이동 모드·보행 상태·자세 등 애니메이션에 필요한 모든 값을 갱신합니다. PoseSearch(Motion Matching)를 위한 궤적 데이터를 생성하고, IsMoving/IsStarting/IsPivoting 등 이동 분석 함수를 블루프린트에 노출합니다. IAO_TraversalTransform 인터페이스를 구현하여 트래버설 어빌리티에서 목표 트랜스폼을 설정/읽을 수 있습니다.
변수명타입지정자설명
Proxy FAOAnimInstanceProxy Transient 스레드 안전 프록시. Owner, Character, MovementComponent 참조를 보관합니다.
CharacterTransform / CharacterTransformLastFrame FTransform BPReadOnly 현재 및 이전 프레임의 캐릭터 월드 트랜스폼입니다.
Velocity / VelocityLastFrame / VelocityAcceleration FVector BPReadOnly 현재 속도, 이전 프레임 속도, 속도의 변화율(가속도). 이동 분석 함수에서 참조됩니다.
Speed2D / bHasVelocity float / bool BPReadOnly 2D 수평 속도 크기와 유효 속도 존재 여부(5.f 이상일 때 true)입니다.
Acceleration / AccelerationAmount / bHasAcceleration FVector / float / bool BPReadOnly 현재 가속도 벡터, 정규화된 가속량(0~1), 가속 존재 여부입니다.
MovementMode / MovementState / Gait / Stance ECustomMovementMode / EMovementState / EGait / EStance BPReadOnly 이동 모드(OnGround/InAir), 이동 상태(Idle/Moving), 보행(Walk/Run/Sprint), 자세(Stand/Crouch). 각각 LastFrame 변수도 존재합니다.
Trajectory / TrajectoryData_Idle / TrajectoryData_Moving FTransformTrajectory / FPoseSearchTrajectoryData BPReadOnly PoseSearch용 궤적 결과와 정지/이동 시 궤적 생성 파라미터입니다.
TrjPastVelocity / TrjCurrentVelocity / TrjFutureVelocity FVector BPReadOnly 궤적에서 추출한 과거(-0.3~-0.2s), 현재(0~0.2s), 미래(0.4~0.5s) 속도 벡터입니다.
CurrentSelectedDatabase / CurrentDatabaseTags UPoseSearchDatabase* / TArray<FName> BPReadWrite 현재 선택된 PoseSearch 데이터베이스와 해당 데이터베이스의 태그 목록입니다.
TraversalTransform FTransform BPReadWrite 트래버설 어빌리티에서 설정한 목표 트랜스폼. IAO_TraversalTransform 인터페이스를 통해 접근됩니다.
HeavyLandSpeedThreshold float BPReadOnly 강한 착지(Heavy Land) 판정 속도 임계값. 기본값 700.f이며, LandVelocity.Z 절대값과 비교합니다.

IsMoving

IsStarting

ShouldSpinTransition
Speed2D가 150.f 이상이고 Pivots 태그가 없을 때 true를 반환합니다. 스핀 전환 애니메이션 조건입니다.
JustTraversal() / JustLanded_Heavy() / JustLanded_Light() BlueprintPure ▶
JustTraversal
DefaultSlot이 비활성이고 MovingTraversal 커브 값이 0 초과이며 궤적 회전 각도가 50도 미만일 때 true를 반환합니다.
JustLanded_Heavy
Character.bJustLanded가 true이고 LandVelocity.Z 절대값이 HeavyLandSpeedThreshold 이상일 때 true를 반환합니다.
JustLanded_Light
Character.bJustLanded가 true이고 LandVelocity.Z 절대값이 HeavyLandSpeedThreshold 미만일 때 true를 반환합니다.

사용처

UAO_PlayerSoundSubsystem
부모: UGameInstanceSubsystem
역할
플레이어 캐릭터의 사운드(발소리, 피격음 등)를 캐릭터 메시 타입에 따라 조회하는 전역 서브시스템
부모 클래스
UGameInstanceSubsystem (GameInstance 수명 동안 유지)
실행 위치
모든 클라이언트 및 서버. 사운드 재생은 로컬에서만 수행
주요 책임
DeveloperSettings에서 SoundDataAsset 로드, 메시 타입+GameplayTag로 사운드 조회, 기본 사운드 폴백 제공
UGameInstanceSubsystem을 상속하여 GameInstance와 같은 수명 동안 유지됩니다. Initialize에서 UAO_PlayerSoundSettings(DeveloperSettings)로부터 CharacterSoundDataAsset과 DefaultSounds를 동기 로드합니다. GetSound/GetSoundFromActor를 통해 캐릭터 메시 타입(ECharacterMesh)과 GameplayTag 조합으로 사운드를 조회하며, DataAsset에서 찾지 못하면 DefaultSounds 맵에서 폴백 사운드를 반환합니다.
변수명타입지정자설명
LoadedDA TObjectPtr<UAO_PlayerSoundDataAsset> Transient Initialize에서 로드된 캐릭터 사운드 데이터 에셋입니다. 메시 타입별 사운드 매핑을 보유합니다.
LoadedDefaultSounds TMap<FGameplayTag, TObjectPtr<USoundBase>> Transient GameplayTag를 키로 하는 기본 사운드 맵입니다. DataAsset에서 사운드를 찾지 못했을 때 폴백으로 사용됩니다.

흐름

UAO_CameraManagerComponent
부모: UActorComponent
역할
캐릭터 상태(사망, 스프린트, 트래버설)에 따라 카메라 프로필을 전환하는 매니저 컴포넌트
부모 클래스
UActorComponent (BlueprintSpawnableComponent)
실행 위치
로컬 클라이언트. TickComponent에서 매 프레임 카메라 값을 보간
주요 책임
GAS 태그 감시, 카메라 상태 Push/Pop, 우선순위 기반 프로필 선택, SpringArm/Camera 값 보간 적용
UActorComponent를 상속하며, GAS(GameplayAbilitySystem)의 GameplayTag 변화를 감시하여 카메라 상태를 자동 전환합니다. ProfileDB(UAO_CameraProfile 데이터 에셋)에서 태그별 카메라 설정(암 길이, 소켓 오프셋, FOV, 래그 등)을 읽고, Requests 맵으로 활성 상태를 관리합니다. 매 프레임 TickComponent에서 우선순위가 가장 높은 프로필을 선택하고, FInterpTo로 부드럽게 보간하여 SpringArm과 Camera에 적용합니다.
변수명타입지정자설명
ProfileDB TObjectPtr<UAO_CameraProfile> EditAnywhere 태그별 카메라 설정(암 길이, FOV, 래그 등)을 보유하는 데이터 에셋입니다.
SpringArm / Camera TObjectPtr<USpringArmComponent> / TObjectPtr<UCameraComponent> (Private) BindCameraComponents로 설정되는 카메라 시스템 참조입니다.
ASC TObjectPtr<UAbilitySystemComponent> (Private) BindToASC로 설정되는 GAS 컴포넌트 참조입니다. 태그 이벤트를 구독합니다.
Requests TMap<FGameplayTag, FAO_CameraRequest> (Private) 현재 활성/페이드 중인 카메라 요청 맵입니다. Tag, Priority, Weight, BlendIn/Out, bActive를 보유합니다.
CurArmLength / CurSocketOffset / CurFOV float / FVector / float (Private) 현재 보간 중인 카메라 세팅 값입니다. 매 프레임 목표 프로필을 향해 FInterpTo로 갱신됩니다.
State_Dead / State_Sprint / State_Traversal FGameplayTag (Private) 캐릭터 상태 태그. 생성자에서 RequestGameplayTag로 초기화됩니다.
Camera_Default / Camera_Dead / Camera_Sprint / Camera_Traversal FGameplayTag (Private) 카메라 프로필 태그. ProfileDB에서 해당 태그로 설정을 검색합니다.
TagEventHandles TArray<FDelegateHandle> (Private) ASC 태그 이벤트 델리게이트 핸들 배열. EndPlay에서 해제합니다.

CachedVOIPTalker의 VoiceLevel을 읽어 발화 상태를 판정합니다. 히스테리시스를 적용하여(시작 >0.02, 종료 ≤0.01) 깜빡임을 방지하고, 위젯에 스피커 아이콘 가시성을 전달합니다.
ShouldHideForSelf() ▶
하는 일
bHideForSelf가 true이고 소유자가 로컬 제어 폰(IsLocallyControlled)이면 true를 반환합니다. 자기 머리 위의 이름표를 숨기는 데 사용됩니다.
방향대상 클래스관계 설명
소유자AAO_PlayerCharacter캐릭터에 부착되어 이름표 위젯을 관리
읽음APlayerState / AAO_PlayerState플레이어 이름을 읽고, 이름이 준비되면 OnPlayerNameReady 델리게이트로 수신
생성UWidgetComponent런타임에 Screen Space WidgetComponent를 생성하여 캐릭터에 부착
참조UAO_NameTagWidget이름 표시/스케일/발화 상태 가시성을 설정하는 위젯 인스턴스
참조UAO_CustomizingComponent캡슐 크기 변경 델리게이트를 구독하여 위젯 높이를 조정
참조UVOIPTalkerSetVOIPTalker로 받아 VoiceLevel을 읽어 발화 상태 판정
