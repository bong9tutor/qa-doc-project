UAO_InventoryComponent
부모: UActorComponent
역할
플레이어 캐릭터에 부착되는 인벤토리 컴포넌트 — 슬롯 기반 아이템 관리
부모 클래스
UActorComponent
실행 위치
서버에서 아이템 픽업·사용·드롭 처리, 클라이언트에는 Slots/SelectedSlotIndex 복제
주요 책임
5슬롯 인벤토리 관리, 아이템 픽업/드롭/사용, 입력 바인딩, 리스너 알림
플레이어 캐릭터에 부착되어 5개 슬롯으로 구성된 인벤토리를 관리합니다. 슬롯 배열(Slots)과 선택 인덱스(SelectedSlotIndex)는 ReplicatedUsing으로 복제되어 클라이언트와 동기화됩니다. Enhanced Input을 통해 슬롯 선택, 아이템 사용(좌클릭), 아이템 드롭(우클릭) 입력을 바인딩합니다. 소모품(Consumable) 사용 시 GAS의 GameplayEffect를 적용하여 체력을 회복하고, 무기(Weapon) 사용 시 라인 트레이스로 대미지를 적용합니다. FInventorySlot 구조체에 ItemID, Quantity, FuelAmount, ItemType 정보를 저장합니다. IAO_InventoryListener 인터페이스를 구현한 리스너에게 슬롯 변경과 선택 변경을 통보합니다. 사망 시 CharDead()로 모든 아이템을 월드에 드롭하며, PlayerState에 인벤토리를 영속 저장할 수 있습니다.

UDataAsset

ItemDataTable 소프트 참조를 에디터에서 설정하고 런타임에 로드
UDataAsset을 상속한 최소한의 데이터 컨테이너입니다. TSoftObjectPtr<UDataTable>로 아이템 데이터 테이블을 소프트 참조하여, 필요할 때만 비동기로 로드할 수 있습니다. 별도의 로직 없이 에디터에서 데이터 테이블 에셋을 지정하는 용도로 사용됩니다.

(함수 없음) ▶

AAO_MasterItem
부모: AAO_BaseInteractable
역할
월드에 배치되는 아이템 액터 — 상호작용으로 인벤토리에 추가
부모 클래스
AAO_BaseInteractable
실행 위치
서버에서 상호작용 처리 및 파괴, 클라이언트에는 ItemID/FuelAmount/ItemTags/CachedItemName 복제
주요 책임
아이템 데이터 적용, 상호작용 시 인벤토리 전달, 물리 시뮬레이션, AI 픽업 지원
AAO_BaseInteractable을 상속하여 플레이어가 상호작용할 수 있는 아이템 액터입니다. ItemID를 기반으로 ItemDataTable에서 메시, 태그, 연료량, 하이라이트 스텐실값을 조회하여 자신에게 적용합니다. MeshComponent에 물리 시뮬레이션이 활성화되어 월드에서 자연스럽게 낙하합니다. InteractionSphere(Pawn 오버랩)로 상호작용 범위를 정의하고, 상호작용 성공 시 Server_HandleInteraction RPC에서 인벤토리에 아이템을 전달한 후 자신을 Destroy()합니다. UAO_PickupComponent를 보유하여 AI(Crab)의 픽업/드롭도 지원합니다.
변수명타입지정자설명
ItemID FName ReplicatedUsing=OnRep_ItemIDEditAnywhereBPReadOnly 아이템 식별자. DataTable 조회 키이자 인벤토리 슬롯에 저장되는 ID. ExposeOnSpawn으로 스폰 시 설정 가능.
ItemDataTable UDataTable* ReplicatedEditAnywhereBPReadOnly FAO_struct_FItemBase 행이 저장된 데이터 테이블. ApplyItemData()에서 아이템 정보를 조회합니다.
FuelAmount float Replicated 연료 아이템일 경우의 연료량. DataTable에서 로드됩니다.
ItemTags FGameplayTagContainer Replicated 아이템 태그 컨테이너. DataTable에서 로드되며, GetOwnedGameplayTags()로 외부에 노출됩니다.
CachedItemName FString Replicated DataTable에서 조회한 아이템 이름. GetInteractionInfo()에서 상호작용 UI 타이틀로 사용됩니다.
InteractionSphere USphereComponent* VisibleAnywhereBPReadOnly 반지름 50의 구체 콜리전. Pawn과 오버랩하여 상호작용 범위를 정의합니다.
PickupComponent UAO_PickupComponent* VisibleAnywhereBPReadOnly AI 픽업/드롭을 위한 컴포넌트. 물리 제어와 부착 로직을 담당합니다.
