Server RPC
ServerMeleeAttack
AO_PlayerCharacter
▶
클라이언트에서 입력 감지 → 서버에서 판정 처리. 인증된 히트만 데미지 적용.
UFUNCTION(Server, Reliable)
void ServerMeleeAttack();

// 구현부 (서버에서 실행)
void AAO_PlayerCharacter::ServerMeleeAttack_Implementation()
{
    // 구체 히트박스 오버랩 감지 후 데미지 적용
    ActivateAbilityByTag(TAG_Ability_Melee);
    MulticastPlayMeleeAnim();
}
Server RPC
ServerInteract
AO_PlayerCharacter
▶
UFUNCTION(Server, Reliable)
void ServerInteract(AActor* Target);

void AAO_PlayerCharacter::ServerInteract_Implementation(AActor* Target)
{
    if (auto* Interactable = Cast<IAO_Interactable>(Target))
    {
        Interactable->Interact(this);
    }
}
Multicast
MulticastPlayMeleeAnim
AO_PlayerCharacter
▶
서버에서 호출 → 모든 클라이언트에서 애니메이션 재생. 시각적 동기화 목적.
UFUNCTION(NetMulticast, Unreliable)
void MulticastPlayMeleeAnim();

void AAO_PlayerCharacter::MulticastPlayMeleeAnim_Implementation()
{
    if (MeleeAnimMontage)
    {
        GetMesh()->GetAnimInstance()->Montage_Play(MeleeAnimMontage);
    }
}
