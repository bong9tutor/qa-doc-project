AI 캐릭터 공통 애니메이션 인스턴스입니다. NativeInitializeAnimation에서 TryGetPawnOwner()를 ACharacter로 Cast하여 Character에 저장하고, Character->GetCharacterMovement()로 MovementComponent를 캐싱합니다.

NativeUpdateAnimation에서는 매 프레임 Super를 호출한 뒤, Character 또는 MovementComponent가 null이면 즉시 리턴합니다. 유효할 경우 Velocity = GetVelocity(), GroundSpeed = Velocity.Size2D()를 계산하고, bShouldMove = (GroundSpeed > 3.0f) && (GetCurrentAcceleration() != ZeroVector)로 이동 여부를 판단합니다. bIsFalling은 MovementComponent->IsFalling()을 직접 캐싱합니다. 모든 변수가 VisibleAnywhere + BlueprintReadOnly이므로 AnimBP의 블렌드 스페이스와 스테이트 머신에서 직접 읽습니다. Category는 "Character"와 "Movement"로 분리되어 있습니다.

AI 캐릭터의 발걸음, 공격 사운드 등을 애니메이션 프레임에 맞춰 재생하는 AnimNotify입니다. 생성자에서 VolumeMultiplier=1.0f, PitchMultiplier=1.0f, MaxDuration=0.0f로 초기화합니다 (헤더의 인라인 초기화와 중복 — 방어적 코딩).

Notify 내부 파이프라인: Super::Notify() 호출 후 MeshComp와 Sound의 null 체크, Owner 액터 획득, AttenuationSettings 분기에서 SpawnSoundAtLocation(UAudioComponent* 반환)과 PlaySoundAtLocation(void)을 사용합니다. GetNotifyName_Implementation()은 Sound 에셋이 있으면 "AI Sound: {에셋명}", 없으면 "AI Sound"를 반환하여 에디터 타임라인에 표시합니다.

bIsCeilingMode — ReplicatedUsing=OnRep_bIsCeilingMode, SetIsReplicatedByDefault(true)
천장 감지
SphereTrace(반경 30) + Normal 각도 체크(MaxCeilingAngle=60도) + 최소 거리 체크
Stalker AI가 천장에 매달려 이동하는 기능을 담당하는 컴포넌트입니다. 생성자에서 bCanEverTick=true, SetIsReplicatedByDefault(true).

핵심 설계: NavMesh는 바닥에만 존재하므로, 캐릭터의 캡슐은 바닥을 Walking 모드로 걸어다니고 GravityScale=0으로 중력만 제거합니다. SkeletalMesh만 RelativeLocation.Z를 천장 높이로 오프셋하여 시각적으로 천장에 붙어 보이게 합니다. Mesh 회전은 FRotationMatrix::MakeFromZX(CeilingNormal, ForwardProjection)으로 계산하고 RInterpTo로 보간합니다.

전환 보간: SetCeilingMode(true) 시 EaseInOut 곡선 (Alpha^2 * (3-2*Alpha))으로 StartOffsetZ에서 TargetOffsetZ까지 보간합니다. 보간 시간은 TransitionInterpSpeed > 0이면 거리/속도로 계산, 0이면 1.03초(40프레임 몽타주 기준 31프레임) 고정값을 사용합니다. TransitionStartDelay(기본 0.1초) 이후부터 보간이 시작됩니다.

클라이언트: OnRep_bIsCeilingMode 콜백에서 UpdateMeshVisualsForCeilingMode()를 호출하여 Mesh 회전(Pitch+180)을 즉시 적용합니다. TickComponent의 else 브랜치에서 클라이언트도 독립적으로 LineTrace하여 Mesh Z 오프셋과 회전을 보간합니다. 천장이 끊기면 서버의 UpdateCeilingPosition에서 SetCeilingMode(false)를 재귀 호출하여 바닥 모드로 자동 전환합니다.
