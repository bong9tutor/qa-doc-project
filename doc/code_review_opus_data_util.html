UAO_GameSettingsManager
부모: UGameInstanceSubsystem
역할
그래픽·해상도·오디오 등 모든 게임 설정을 중앙에서 관리하는 전역 서브시스템
생존 범위
GameInstance 수명과 동일 (GameInstanceSubsystem)
설정 적용 방식
UAO_GameUserSettings를 가져와 설정값 변경 후 ApplyAndSave 패턴
이벤트 알림
OnSettingsApplied 델리게이트로 UI에 적용 완료 알림
UGameInstanceSubsystem을 상속하여 게임 실행 동안 살아있는 전역 설정 매니저입니다. 내부적으로 UAO_GameUserSettings의 정적 함수로 인스턴스를 가져오며(매번 Get 호출 — StalePointer 방지 설계), 그래픽 Scalability(안티앨리어싱, 그림자, 텍스처 등 11개 항목)를 개별적으로 설정할 수 있습니다. 해상도 설정은 정적 배열(16:9, 21:9, 32:9 등 총 11개)에서 인덱스로 선택하며, 오디오는 EAudioType 열거형으로 Master/Music/SFX/UI/Voice/Ambient를 구분합니다.

cpp 구현 분석: Initialize()에서 InitUserSettings()를 호출하여 기존 설정을 로드·적용합니다. ApplyAndSaveAllSettings()는 ApplySettings(false) → ApplyCustomSettings() → OnSettingsApplied.Broadcast() 순으로 실행되며, ApplyResolutionSettings()와 ApplyNonResolutionSettings()는 각각 무거운 해상도 변경과 가벼운 비해상도 변경을 분리합니다. SetToDefaults()는 UAO_DelegateManager의 OnResetAllSettings를 Broadcast하여 UI 리셋도 연동합니다. GetResolutionInfoList()는 static TArray를 사용하여 최초 호출 시에만 배열을 채우는 싱글턴 초기화 패턴입니다. SetAudioVolume()은 FMath::Clamp(0~2)로 볼륨을 제한합니다.

UAO_GameUserSettings
부모: UGameUserSettings
역할
엔진 UGameUserSettings 확장 — 오디오·업스케일링·음성채팅 등 커스텀 설정 저장
저장 방식
UCLASS(config=GameUserSettings)로 모든 Config UPROPERTY가 .ini에 자동 직렬화
싱글턴
엔진이 관리하는 단일 인스턴스. GetGameUserSettings()로 접근 (Cast 기반)
확장 포인트
ApplyCustomSettings() 가상 함수 — 현재 빈 구현, 향후 SoundMix 적용 등 추가 가능
UCLASS(config=GameUserSettings)로 선언되어 모든 Config UPROPERTY가 GameUserSettings.ini에 자동 저장·로드됩니다. 엔진 기본 UGameUserSettings에 없는 오디오 볼륨 6채널, 업스케일링(DLSS/FSR/Built-in), 음성채팅, 언어 옵션 등을 확장합니다.

cpp 구현 분석: GetGameUserSettings()는 UGameUserSettings::GetGameUserSettings()를 Cast<UAO_GameUserSettings>로 변환합니다. DefaultEngine.ini에서 [/Script/Engine.Engine] GameUserSettingsClassName이 이 클래스로 등록되어 있어야 Cast가 성공합니다. 반환 타입이 TObjectPtr<UAO_GameUserSettings>로, UFUNCTION으로 노출하려면 raw pointer로 전환이 필요합니다(주석 기재). ApplyCustomSettings()는 현재 빈 구현이며, 볼륨값을 SoundMix에 적용하거나 입력 감도를 설정하는 로직을 추가할 확장 지점입니다.
