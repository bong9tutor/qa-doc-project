AAO_LobbyInteractable
부모: AAO_BaseInteractable
역할
로비 맵 전용 상호작용 오브젝트 — 레디, 게임 시작, 초대, 옷장
열거형
EAO_LobbyInteractType: ReadyToggle, StartGame, InviteFriends, Wardrobe
권한 체크
ReadyToggle은 게스트만, StartGame은 호스트만 사용 가능
네트워크
Server RPC를 PlayerController_Lobby를 통해 호출
AAO_BaseInteractable을 상속하며 로비 맵에서의 네 가지 상호작용을 InteractType 열거형으로 분기합니다. IsLobbyHostController 유틸리티로 호스트/게스트를 판별하고, ReadyToggle은 로컬 선반영 후 Server_SetReady를 호출합니다. StartGame은 호스트만 Server_RequestStart를 호출할 수 있으며, InviteFriends와 Wardrobe는 모든 플레이어가 사용 가능합니다.

소모성 시스템

AActor와 IAO_Interface_Interactable을 함께 상속하는 월드 배치형 상호작용 베이스 클래스입니다. 홀딩 시작/종료(OnInteractActiveStarted/Ended)와 성공(OnInteractionSuccess)을 K2 이벤트로 Blueprint에 연동합니다. 소모성(bShouldConsume) 상호작용은 성공 시 다른 홀딩 플레이어들의 어빌리티를 GAS 태그로 강제 취소합니다. Transform 애니메이션은 타이머 기반 VInterpTo/RInterpTo로 부드러운 이동/회전을 구현합니다.
