주요 책임
공통 GameMode 타입 제공, 하위 클래스에서 기능 확장
AGameMode를 상속하는 프로젝트 루트 GameMode 클래스입니다. 현재는 추가 멤버나 함수 없이 빈 껍데기로, 모든 맵별 GameMode가 이 클래스를 직접 또는 간접적으로 상속합니다. 공통 로직이 필요할 경우 이 클래스에 추가하면 모든 GameMode에 자동 반영됩니다.
추가 멤버 변수 없음 — AGameMode 기본 속성만 상속합니다.
추가 함수 없음 — AGameMode 기본 가상 함수만 상속합니다.

AAO_GameMode_InGameBase
부모: AAO_GameMode
역할
인게임(Lobby, Stage, Rest) 공통 기반 GameMode
생존 범위
현재 레벨 동안 존재 (SeamlessTravel 시 파괴 후 재생성)
실행 위치
서버 전용 (Authority Only)
주요 책임
SeamlessTravel 활성화, 보이스 채팅 관리, 동기화 ServerTravel, 인벤토리 보존 판정
Lobby, Stage, Rest GameMode가 공통으로 상속하는 중간 기반 클래스입니다. 생성자에서 bUseSeamlessTravel = true로 설정하여 맵 이동 시 연결이 끊기지 않도록 합니다. 보이스 채팅 시작/정지/뮤트 업데이트를 모든 클라이언트에 일괄 전파하는 함수와, ServerTravel 전 모든 플레이어의 오디오 리소스 정리를 기다리는 동기화 메커니즘(RequestSynchronizedServerTravel)을 제공합니다. 또한 맵 이동 직전 인벤토리/체력 보존 판정(HandlePlayerTravel)과 세이프존 체크(CheckPlayerInsideSafeZone) 로직을 수행합니다.
변수명타입지정자설명
CleanupCompletePlayers TSet<TObjectPtr<AAO_PlayerController>> UPROPERTY ServerTravel 동기화 시, 오디오 리소스 정리를 완료한 플레이어 컨트롤러 세트입니다.
bIsTravelSyncInProgress bool (private) 동기화 ServerTravel이 진행 중인지 여부를 나타내는 플래그입니다.
PendingTravelURL FString (private) 동기화 대기 중인 ServerTravel 목적지 URL을 보관합니다.
