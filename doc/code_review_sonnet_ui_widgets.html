UAO_UserWidget
부모: UUserWidget
역할
프로젝트 전체 위젯의 공통 기반 클래스로, 확장성을 위한 공통 기능(UI 사운드, ESC 닫기, 호버 불투명도)을 제공합니다.
부모 클래스
UUserWidget
주요 책임
DataTable 기반 UI 사운드 재생, ESC 닫기 이벤트 인터페이스, 호버/언호버 불투명도 기본값 제공
구조체
FAO_SoundRow (FTableRowBase) - SoundAsset, VolumeMultiplier, PitchMultiplier, Description
프로젝트 내 모든 커스텀 위젯의 최상위 부모로 설계된 클래스입니다. WBP_Base_Modal 등 중간 블루프린트 계층을 통해 Open/Close 애니메이션 등 공통 행동을 하위 WBP에 일괄 적용할 수 있습니다. PlayUISoundFromDataTable 함수는 DataTable에서 사운드 행을 찾아 PlaySound2D로 UI 사운드를 재생합니다. OnEscapeCloseRequested는 BlueprintImplementableEvent로, 블루프린트에서 ESC 키 닫기 동작을 구현할 수 있습니다.

UAO_HostDialogWidget
부모: UUserWidget
역할
방 이름과 비밀번호를 입력받아 HostSessionAuto를 호출하는 마우스 중심 모달 팝업 다이얼로그
부모 클래스
UUserWidget (UAO_UserWidget을 상속하지 않음)
주요 책임
방 이름 유효성 검사(1~10자), 비밀번호 4자리 제한, OnlineSessionSubsystem을 통한 세션 생성
입력 모드
FInputModeUIOnly로 마우스 커서만 표시, 방 이름 텍스트박스에 포커스 설정
방 생성(Host) 팝업 다이얼로그입니다. NativeConstruct에서 Backdrop 클릭 바인딩, 텍스트 변경 콜백, 모달 입력 모드를 설정합니다. 방 이름은 1~10자, 제어 문자(0x20 미만) 불가로 유효성 검사하며, 비밀번호는 4자리까지 제한합니다. OnClicked_Create에서 유효성 검사를 통과하면 OnlineSessionSubsystem의 HostSessionAuto(4, RoomName, Password)를 호출하고 팝업을 제거합니다. Enter 키로 방 이름 입력을 커밋하면 OnClicked_Create가 즉시 실행됩니다.
변수명타입지정자설명
Txt_RoomName UEditableTextBox* BindWidget 방 이름 입력 텍스트박스입니다.
Txt_Password UEditableTextBox* BindWidget 비밀번호 입력 텍스트박스입니다. 4자리 제한이 적용됩니다.
Btn_Create UUserWidget* BindWidget 방 생성 버튼입니다. 방 이름 유효성에 따라 Enable/Disable이 갱신됩니다.
Btn_Cancel UUserWidget* BindWidget 취소 버튼으로 팝업을 닫습니다.
Btn_Backdrop UButton* BindWidget 배경 클릭 시 Cancel과 동일하게 팝업을 닫습니다.

하는 일

RebuildBoard는 GameState.PlayerArray에서 AO_PlayerState를 순회하여 호스트/게스트 여부, 레디 상태를 수집하고, 입장 순서(JoinOrder)로 정렬하되 호스트를 항상 맨 위에 배치합니다. IsEveryoneReadyExceptHost로 호스트 제외 전원 레디 여부를 확인하여 호스트의 "Start" 활성 상태를 결정합니다.
